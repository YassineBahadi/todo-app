# Todo Task Manager 📝

Un gestionnaire de tâches moderne avec Angular 19 et Spring Boot 3.x.

## 🚀 Fonctionnalités

- ✅ CRUD complet des tâches
- 🔐 Authentification JWT
- 🔍 Recherche et filtres
- 📱 UI Responsive
- 🐳 Dockerisé

## 🛠️ Stack Technique

### Frontend
- Angular 19 (Standalone Components)
- Angular Material
- Reactive Forms

### Backend  
- Spring Boot 3.x
- Spring Security + JWT
- Spring Data JPA
- PostgreSQL
